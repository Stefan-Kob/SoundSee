 - [ ] **User accounts**
	 - [ ] Account management
	 - [ ] Profiles - Private and Public
	 - [ ] Following profiles
 - [ ] **Messaging (text and image/video)**
 - [ ] **Posts**
	 - [ ] Likes/dislikes
	 - [ ] Stories
 - [ ] **App customization**
 - [ ] **Home Page**
	 - [ ] Profile page
	 - [ ] Notifications page
	 - [ ] Messaging page

	 - [ ] Create post page
